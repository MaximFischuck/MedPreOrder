# MedPreOrder##  О проекте

MedPreOrder - это полнофункциональный интернет-магазин лекарственных препаратов, разработанный для удобного онлайн-покупки медикаментов. Проект включает в себя каталог товаров, корзину покупок, систему оформления заказов с валидацией данных и адаптивный дизайн.

Цели проекта:
- Создание удобной платформы для покупки лекарств онлайн
- Реализация полного цикла покупки (от выбора до оформления)
- Обеспечение безопасности данных пользователей
- Создание адаптивного интерфейса для всех устройств

##  Функциональность

### Основные возможности:
- *Полный каталог товаров с фильтрацией по категориям
- Детальная карточка товара с полной информацией о препарате
- Интеллектуальная корзина с управлением количеством
- Система оформления заказов с валидацией данных
- Адаптивный дизайн для всех устройств
- Локальное хранение данных (корзина, заказы)
- Уведомления о действиях пользователя

### Дополнительные функции:
- Быстрый поиск по каталогу 
- Информация о рецептурных препаратах
-  Расчет стоимости доставки в реальном времени
-  Валидация форм (ФИО, телефон, email)
